# 🐦 Flappy PisPis

> *"PisPis"* means **bird** in Bisaya.

An Arduino Uno R3 handheld game inspired by Flappy Bird! Guide your little **PisPis** through obstacles and survive for as long as possible.

Unlike most Arduino games, **Flappy PisPis** isn't silent. It features a buzzer that plays background music (it's kinda loud 😂), sound effects for different events, and a game-over tune when your bird crashes.

As your score increases, the obstacles become progressively harder, making every run more challenging than the last!

---

# 🎮 Features

* 🐦 Fun Flappy Bird-inspired gameplay
* 🎵 Background music using a buzzer
* 🔊 Sound effects for jumping, game over, and other events
* 📈 Dynamic difficulty (higher score = harder obstacles)
* 🏆 Displays your final score after every game
* ⚡ Runs entirely on an Arduino Uno R3

---

# 🛠️ Materials Used

* Arduino Uno R3 (of course 😎)
* 128×64 OLED Display (SH1106)

  ```cpp
  U8G2_SH1106_128X64_NONAME_F_HW_I2C u8g2(U8G2_R0);
  ```
* Small breadboard
* Solid-core jumper wires
* 2 push buttons
* 1 passive buzzer
